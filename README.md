# Превью социальной сети

Это превью-страница для новой социальной сети, разработанной для демонстрации основных функций и возможностей платформы.

## Описание

Социальная сеть предлагает следующие возможности:
- Загрузка и отображение изображений в профиле
- Общение с другими пользователями
- Встроенный магазин
- Классические игры для развлечения
- Публикация новостей и постов

## Структура проекта

- `index.html` - Основная HTML-страница с слайд-шоу превью
- `index.css` - Стили для страницы
- `1.png` до `6.png` - Изображения для слайд-шоу

## Просмотр

Откройте `index.html` в браузере для просмотра превью.
